Each section repeats the same loop: fill → submit → under review → ✅ validated. If the admin finds an issue, a correction appears; you fix it and resubmit. You chat with the admin in EcoLoop throughout.

What's true for you

›You see only your own data. Every request is scoped to your caterer id.
›Your bank details are encrypted. After you submit, you only see a masked last-4.
›Smart Import never auto-fills. You confirm or edit every suggested value.
›You propose; the admin decides. You can't price modules or approve your own go-live.
